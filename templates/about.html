The Core Problem

Traditional visitor logs are a major security and privacy loophole. Manual entries allow unauthorized data visibility and are easy to tamper with. They lack data integrity and are difficult to audit during security breaches.

Our Solution

QR-Secure digitizes the entire workflow. We encrypt sensitive visitor data using AES-256 and issue secure, time-bound QR passes. Data like phone numbers and purpose are secured end-to-end with local SQLite storage.

Liveness Innovation

To prevent photo-spoofing, we implemented an Eye-Blink detection system. Registration only proceeds when a real human is verified through a 2-blink liveness check using real-time EAR (Eye Aspect Ratio) tracking.
